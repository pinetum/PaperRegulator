# Paper Regulator

 * 自動掃描剪貼簿內容作為文字來源
 * 移除段落中換行時的英文連接符號
 * 即時連結google translate顯示翻譯結果

#### 僅需要對想要翻譯的文字按下複製

### Installation

#### Windows

[Bin_paperRegulator_0.1_x64.zip](https://github.com/pinetum/PaperRegulator/raw/master/bin/paperRegulator_0.1_win64.zip)


#### linux or os X

requirement:

* g++
* wxWidget 2.9+
* codelite (optional)
```bash
make
./Release/PaperRegulator
```
